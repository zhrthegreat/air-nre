#民航系统实体关系抽取

##说明
本工程使用于民航系统的实体关系抽取，数据资料来源于2011-2018民航统计公报及民航资源网实体资料[http://data.carnoc.com/corp/]

关系抽取模型训练使用清华大学THUNLP的opennre开源工具包，基本实现民航系统的抽取，且有良好效果

实体识别调用jieba开源工具，并在内容上进行了一些修改

##环境配置

要求：pytorch>=1.3.0

其他安装包依赖见requirements.txt

`pip install -r requirements.txt`

若对数据集有改动需求参考 **模型训练** 小节

##关系抽取模型训练

原数据须满足“实体1 实体2 关系 原句”的格式,保存在root/dataset/data.txt

运行代码
```
cd root/dataset

python gen.py
```

生成air_all.txt数据,可自分配数据集

运行train_air_bert_sotfmax.py文件(参数调节见源文件)
```
cd ..

python train_air_bert_sotfmax.py
```

通过自制数据集获得训练结果
